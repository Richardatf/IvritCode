# Instruction set

The authoritative instruction registry is `packages/core/src/instructions.ts`. It defines all 22 technical transformations, names, indices, and Aleph Olam access. All reads for an operation come from its original input state. Vav performs pairwise bitwise conjunction. Plain Yod rotates the existing pattern by Aleph Olam and adds a destination phase; `יּ` is the explicit unification form. Plain Tav preserves the completed state as a seal checkpoint; `תּ` seals and explicitly halts.
