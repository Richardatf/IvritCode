# Niqqud

Version 1 recognizes core vowel marks, Dagesh, Shin Dot, and Sin Dot. Marks survive parsing, disassembly, and traces. They are neutral in permissive execution unless an explicit marked behavior is defined. Strict mode rejects unimplemented semantics with a suggested correction. Yod with Dagesh deliberately unifies the visible registers to Aleph Olam; Tav with Dagesh seals and explicitly halts.
