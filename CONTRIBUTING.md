# Contributing to IvritCode

Use Node 22, create a focused branch, and keep technical semantics separate from interpretation. Run `npm install`, `npm run format`, `npm run lint`, `npm run typecheck`, `npm test`, and `npm run build` before submitting a pull request. New instruction behavior requires documentation, targeted mathematical tests, immutability checks, and Unicode examples where relevant. Never commit credentials or label unverified strings as Hebrew words.
